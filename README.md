# Pi-llm TypeScript Server

A basic Node.js TypeScript project structure, configured for development and building.

## Project Structure

- `src/index.ts` - Server entry point.
- `dist/` - Compiled output directory.
- `tsconfig.json` - TypeScript compiler configuration.
- `package.json` - Dependencies and script definitions.

## Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run in Development Mode (with Live Reload/Watching)
Runs the server directly and watches for changes:
```bash
npm run dev
```

### 3. Build for Production
Compiles the TS code into Javascript in the `dist` directory:
```bash
npm run build
```

### 4. Run Build
Runs the compiled server:
```bash
npm run start
```
